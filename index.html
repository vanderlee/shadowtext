jQuery ShadowText widget

This jQuery widget gives each individual text symbol a dropshadow, moving in response to the mouse position.

Demonstrations

Default

$('.demo-default').shadowtext();
ShadowText

Advanced options

$('.demo-advanced').shadowtext({
	color:			"#800",
	distance: 		'100px',
	blurClose:		0,
	blurFar:		'100px',
	opacityClose:	1,
	opacityFar:		0,
	framerate:		25
});
ShadowText

Focus

$('.demo-focus').shadowtext({
	color:			'black',
	distance: 		0,
	blurClose:		0,
	blurFar:		'100px',
	hideText:		true,
	mouseRange:		'400px'	// requires jQuery.px
});
Move mouse to focus!

Axis

$('.demo-axis').shadowtext({
	distance:	'1em',
	axis:		'y',
	mouseRange:	'50vh'
});
