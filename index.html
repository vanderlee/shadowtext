jQuery ShadowText widget

This jQuery widget gives each individual text symbol a dropshadow, moving in response to the mouse position.

Demonstrations

Default

$('.demo-default').shadowtext();
Default
options

Background

$('.demo-background').shadowtext({
	color:			"#800",
	distance: 		100,
	blurFar:		100,
	opacityFar:		0,
	framerate:		25,
	mouseRange:		1000
});
Background

Focus

$('.demo-focus').shadowtext({
	color:			'black',
	distance: 		0,
	blurFar:		100,
	hideText:		true,
	mouseRange:		400
});
Move mouse to focus

Axis

$('.demo-axis').shadowtext({
	distance:	'1em',		// requires jQuery.px
	axis:		'y',
	mouseRange:	'50vh'		// requires jQuery.px
});
Up|Down
